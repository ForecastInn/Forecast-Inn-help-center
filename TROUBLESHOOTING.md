# Troubleshooting Guide - ForecastInn App
# Solusi Masalah Umum Saat Pakai ForecastInn

---

## 🔧 Login Issues

### Masalah: **Lupa Password**

**Solusi:**
1. Di halaman login, klik **"Forgot password?"**
2. Masukkan email terdaftar
3. Cek email inbox (termasuk spam folder)
4. Klik link reset password
5. Masukkan password baru
6. Login dengan password baru

**Masih tidak bisa?**
- Hubungi support: support@forecastinn.com

---

### Masalah: **Email Tidak Diterima**

**Kemungkinan:**
- Email salah ketik
- Email masuk ke spam folder
- Firewall office memblokir

**Solusi:**
1. Cek folder **Spam / Junk**
2. Tambahkan **support@forecastinn.com** ke contacts
3. Minta admin IT untuk whitelist forecastinn.com

---

### Masalah: **"Invalid Email or Password"**

**Solusi:**
1. Pastikan **email benar** (cek typo)
2. Pastikan **Caps Lock tidak aktif**
3. Coba **copy-paste** password dari notes
4. Reset password kalau masih tidak bisa

---

## 📊 Dashboard Issues

### Masalah: **Data Tidak Muncul / Blank**

**Solusi:**
1. **Refresh browser** (F5 atau Ctrl+R)
2. **Clear browser cache:**
   - Chrome: Ctrl+Shift+Del → Clear cache
   - Firefox: Ctrl+Shift+Del → Clear cache
3. Cek **internet connection**
4. Coba **browser lain** (Chrome/Firefox/Safari/Edge)
5. **Logout → Login kembali**

---

### Masalah: **Forecast Lama Load**

**Normal:**
- Forecast 7-14 hari: **5-10 detik**
- Forecast 30 hari: **15-30 detik**

**Solusi jika lebih dari 1 menit:**
1. Refresh halaman
2. Cek internet speed
3. Coba date range lebih kecil dulu
4. Hubungi support kalau sering terjadi

---

### Masalah: **Data Tidak Update**

**Solusi:**
1. Cek **Last Updated** timestamp di dashboard
2. Data biasanya update **setiap 1 jam**
3. Kalau lebih dari 2 jam tidak update:
   - Logout → Login
   - Hubungi support

---

## 💻 Browser / Technical Issues

### Masalah: **Blank Screen / White Screen**

**Solusi:**
1. Pastikan **JavaScript enabled** di browser
2. Update browser ke **versi terbaru**
3. Coba **Incognito/Private mode**
4. Clear cache & cookies
5. Disable browser extensions (AdBlock, dsb)

---

### Masalah: **Tidak Bisa Klik Tombol / Menu**

**Solusi:**
1. Refresh browser
2. Coba browser lain
3. Pastikan **tidak ada popup blocker**
4. Cek console error (F12 → Console tab)
5. Hubungi support dengan screenshot error

---

### Masalah: **Mobile View Tidak Responsive**

**Solusi:**
1. Update browser di HP
2. Cek **internet connection**
3. Coba **landscape mode**
4. Gunakan browser: Chrome / Safari (bukan built-in browser)

---

## 📱 Data Sync Issues

### Masalah: **Booking Tidak Sama dengan PMS**

**Kemungkinan:**
- Delay sync dari PMS ke ForecastInn
- Configuration salah

**Solusi:**
1. Tunggu **5-10 menit** (sync delay normal)
2. Cek **integration status** di Settings
3. Hubungi IT / PMS provider
4. Hubungi ForecastInn support

---

### Masalah: **Revenue Beda dengan Report Internal**

**Solusi:**
1. Cek **date range** yang sama
2. Pastikan **currency** yang sama
3. Include/exclude **cancelled bookings**?
4. Compare **line by line** di detail report
5. Hubungi support kalau selisih signifikan

---

## 🚨 Performance Issues

### Masalah: **App Sangat Lambat**

**Solusi:**
1. Cek **internet speed** (min 5 Mbps)
2. **Close other tabs** di browser
3. **Clear browser cache**
4. Coba **off-peak hours** (pagi/siang)
5. Hubungi support kalau persist

---

### Masalah: **Browser Crash / Freeze**

**Solusi:**
1. Update browser ke versi terbaru
2. Increase **RAM allocation** (kalau low RAM)
3. Disable **heavy extensions**
4. Coba **lightweight browser** (Brave, Edge)
5. Hubungi support untuk alternative access
